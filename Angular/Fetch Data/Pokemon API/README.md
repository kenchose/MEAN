#Objective

Using the Pokemon API (https://pokeapi.co/) do the following:
1. Make an API request from your service to get data about your favorite Pokemon
2. Print the most interesting data about your Pokemon in a string
3. Using the data from your first API call, construct another API call
4. Print how many Pokemon share your favorite Pokemon's abilities
